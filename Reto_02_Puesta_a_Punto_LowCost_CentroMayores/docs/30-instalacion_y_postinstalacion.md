# 30 — Instalación y post-instalación

## Pasos de instalación (resumen)

1. Arranque desde USB con Linux Mint Cinnamon en modo instalación.
2. Selección de idioma, zona horaria y distribución de teclado.
3. Instalación limpia del sistema en el SSD, usando el particionado automático.
4. Finalización de la instalación y primer arranque del sistema.

## Paquetes esenciales

- **Navegador web:** Firefox (incluido por defecto).
- **Ofimática online:** acceso mediante navegador a Google Docs / Microsoft 365.
- **Multimedia y codecs:** instalación de codecs multimedia durante la instalación para asegurar reproducción de audio y vídeo.

## Usuarios, contraseñas y políticas de actualización

- Creación de un usuario estándar con contraseña sencilla y recordable para el centro.
- Cuenta de administrador reservada para mantenimiento.
- Actualizaciones automáticas activadas solo para seguridad y estabilidad, evitando cambios bruscos en la interfaz.
