# 50 — Optimización de rendimiento

## Memoria y swap

- **zRAM / swapfile:** activar zRAM y configurar swapfile de tamaño moderado (ej. 2–4 GB) para mejorar multitarea ligera sin saturar RAM.

## Servicios y procesos

- **Deshabilitar servicios innecesarios:** desactivar demonios y servicios que no se utilizan en el centro para liberar recursos.

## Navegador

- Ajustes de bajo consumo: bloqueo de anuncios, limitar memoria caché, desactivar flags innecesarios para mejorar rendimiento de navegación.

## Limpieza de temporales

- Borrar archivos temporales, cachés y logs antiguos periódicamente para mantener espacio libre y evitar ralentizaciones.
