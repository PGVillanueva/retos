# 20 — Selección de SO ligero

Candidatos (ejemplos): Linux Mint XFCE, Xubuntu, Debian + XFCE, Zorin Lite, Tiny11 (si procede).

| Distro | Requisitos | Soporte | Pros | Contras | Decisión |
|---|---|---|---|---|---|
| Linux Mint XFCE      | CPU x86_64, 2 GB RAM (4 GB recomendado), 20 GB de almacenamiento  | LTS (~5 años) | Muy ligero, estable, ideal para hardware antiguo  | Interfaz menos moderna  | No |
| Xubuntu  | CPU x86_64, 2 GB RAM (4 GB recomendado), 20 GB de almacenamiento         | LTS (3–5 años según versión)   | Buen equilibrio entre ligereza y usabilidad  | Menor enfoque en usuarios novatos   | No  |
| Debian + XFCE        | CPU x86_64, 1–2 GB RAM, 10–20 GB de almacenamiento  | Muy largo (≈5 años + LTS)  | Extremadamente estable y ligero  | Instalación y mantenimiento menos amigables  | No  |
| Zorin OS Lite  | CPU x86_64, 2 GB RAM, 20 GB de almacenamiento  | Medio (según versión) | Diseño muy orientado a ex-usuarios de Windows  | Menor soporte comunitario que Mint  | No | 
| Tiny11  | CPU x86_64, 4 GB RAM, 20 GB de almacenamiento   | No oficial  | Compatibilidad con software Windows  | No oficial, posibles problemas legales y de soporte  | No|
| Linux Mint Cinnamon | CPU x86_64, 2 GB RAM (4 GB recomendado), 20 GB de almacenamiento| Soporte LTS (~5 años por versión)| Interfaz muy intuitiva, similar a Windows, gran comunidad y estabilidad | Entorno Cinnamon algo más pesado que XFCE en equipos muy antiguos  | Sí |

**Justificación final:**  
Se elige Linux Mint con Cinnamon por ofrecer una experiencia de usuario sencilla y familiar para personas mayores, buen equilibrio entre rendimiento y usabilidad, y un soporte prolongado que garantiza estabilidad y actualizaciones de seguridad.
