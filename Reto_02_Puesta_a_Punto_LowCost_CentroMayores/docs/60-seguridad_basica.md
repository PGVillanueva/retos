# 60 — Seguridad básica

## Control de usuarios y privilegios

- Crear **usuario estándar** para uso diario, sin privilegios de administrador.
- Acceso a **sudo bajo demanda** solo para tareas de mantenimiento o configuración avanzada.

## Actualizaciones y navegación

- Activar **actualizaciones automáticas de seguridad** para sistema y paquetes.
- Configurar navegación segura mediante **listas blancas o filtros** si se considera necesario para los usuarios.

## Recuperación y respaldo

- Configurar **restauración rápida** mediante snapshots del sistema o backups base con rsync, permitiendo revertir cambios o fallos de manera sencilla.
